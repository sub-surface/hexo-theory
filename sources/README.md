# Sources

Source material and imported research bundles live here.

- `literature/`: PDFs and source notes.
- `bundles/`: original zip archives.
- `external-runs/`: extracted bundles, generated atlases, and external code used
  as provenance for claims in `SPEC.md` and `docs/theory/`.

Do not treat `external-runs/` as active project code unless a file has been
explicitly promoted into `engine/`, `experiments/`, or `competition/`.
