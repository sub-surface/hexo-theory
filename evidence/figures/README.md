# Figures Directory

`evidence/figures/` contains tracked PNG summaries generated by scripts in
`experiments/`. A figure should be reproducible from a result file whenever
possible.

Naming convention:

- `fig_<experiment>.png` for one primary figure;
- `fig_<experiment>_<view>.png` for multiple views of the same result.

Current important CGT figures:

- `fig_cgt_hackenbush_summary.png`: hot-move behavior, component count,
  potential/temperature correlation, and heat concentration.
- `fig_cgt_hackenbush_scatter.png`: per-position scatter showing that potential
  mass and CGT urgency are related but not identical.
- `fig_cgt_sequences.png`: Sloane-style sequence trajectories for live lines,
  D6 candidate orbits, hot components, and thermal entropy.
- `fig_surreal_fractal_strategy.png`: verified recursive strategy pattern on the
  HeXO lattice, colored by recursion level.
- `fig_surreal_fractal_shells.png`: log-scale branching profile for the
  surreal/fractal strategy generator.
- `fig_crystal_survey_gallery.png`: real-space sample gallery across agents,
  controls, Rust pure-MCTS, and recursive fractal construction.
- `fig_crystal_survey_metrics.png`: normalized heatmap of crystal observables.
- `fig_crystal_survey_harmonics.png`: angular harmonic plot for symmetry
  breaking.
- `fig_crystal_survey_diffraction.png`: Bragg99 reciprocal-space comparison.
- `fig_crystal_survey_fractal_highres.png`: high-resolution verified strategy
  fractal generated during the survey.
